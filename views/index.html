API Documentation

Available Endpoints

Here are some of the available endpoints in this template:

GET /

Root endpoint (this page)

Returns the welcome page for the API server.

POST /api/v1/admin

Register a new admin user with role-based permissions.

Requires username, password in the request body.

PATCH /api/v1/admin

Register a new admin user with role-based permissions.

Requires username, password in the request body.

DELETE /api/v1/admin

Register a new admin user with role-based permissions.

Requires username, password in the request body.

POST /api/v1/admin/auth

Authenticate admin user and receive JWT token.

Requires username and password in the request body.

Installation Guide

1 Clone the repository

git clone https://github.com/DevBigEazi/nodejs-app-template.git
cd NodeJS-MVC-App-Template

2 Install dependencies

npm install
# or
yarn install

3 Create environment variables

Create a .env file in the root directory with the following contents:

PORT = 8500
DATABASE_URI = mongodb://localhost:27017/your_database
ACCESS_TOKEN_SECRET = your_secret_key
CLOUDINARY_CLOUD_NAME = your_cloudinary_name
CLOUDINARY_API_KEY = your_cloudinary_api_key
CLOUDINARY_SECRET_KEY = your_cloudinary_secret_key

4 Start the development server

npm run dev
# or
yarn dev
